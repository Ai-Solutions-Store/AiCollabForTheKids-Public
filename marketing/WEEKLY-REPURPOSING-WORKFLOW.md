### TUESDAY 8:00am - 5:00pm: TRANSFORMATION SPRINT
**Goal:** Transform blog into 4 major formats (thread, video script, podcast script, email)

#### TUESDAY 8:00am-9:00am: Thread Extraction (60 min)
**Task:** Convert blog into Twitter thread

**Process:**
1. Open blog post in Google Docs
2. Read through once (identify 5-7 key insights)
3. Create new Google Doc: "[Blog Title] - Twitter Thread"
4. Extract structure:
```
Tweet 1: HOOK (Problem/stat that grabs attention)
Tweet 2: CONTEXT (Why this matters)
Tweet 3: INSIGHT 1 (Statement + supporting fact)
Tweet 4: INSIGHT 1 (continued/example)
Tweet 5: INSIGHT 2 (Statement + supporting fact)
Tweet 6: INSIGHT 2 (continued/example)
Tweet 7: INSIGHT 3 (Statement + supporting fact)
Tweet 8: INSIGHT 3 (continued/example)
Tweet 9: MISSION (FOR THE KIDS 60% to charities)
Tweet 10: CTA (Read full post: [link])
```

5. Write each tweet (280 chars max, test in Buffer)
6. Add emojis (1-2 strategic uses)
7. Save in Google Doc
8. Share with team for review

**Quality check:**
```
‚òê Each tweet stands alone if needed
‚òê Thread flows logically
‚òê CTA clear and linked
‚òê No repeated points
‚òê Stats accurate
‚òê Tone matches brand
‚òê First tweet hooks readers
‚òê Last tweet drives action
```

**Time estimate:** 60 minutes for complete thread

#### TUESDAY 9:00am-10:30am: Video Script (90 min)
**Task:** Write script for 5-7 minute YouTube video

**Process:**
1. Open new Google Doc: "[Blog Title] - Video Script"
2. Choose video angle:
   - **List:** "5 insights from [blog topic]"
   - **Story:** "Here's what happened when we [action]"
   - **How-to:** "Here's how to [achieve result]"
   - **Debate:** "Here's why [common belief] is wrong"
   - **Case study:** "We tried [thing], here's what happened"

3. Write script with timing:
```
INTRO (0-15 sec) [~40 words]
[Hook that makes people stop scrolling]

INSIGHT 1 (15-90 sec) [~200 words]
[Statement]
[Story or example]
[Why it matters]

INSIGHT 2 (90-180 sec) [~200 words]
[Repeat pattern]

INSIGHT 3 (180-270 sec) [~200 words]
[Repeat pattern]

OUTRO (270-300 sec) [~80 words]
[Summary]
[CTA]
[Sign off]

TOTAL: ~5-7 minutes
```

4. Add visual cues:
   - [B-Roll: Talking while walking]
   - [Graphic: Stat overlay]
   - [Text: Key insight appears on screen]
   - [Cut to: Expert testimonial]

5. Save in Google Doc
6. Share for review

**Script writing tips:**
- Write conversational (how you'd talk, not write)
- Use "you" and "we" (personal, not corporate)
- Short sentences (for delivery)
- Include 2-3 pauses for breath (written as "...")
- Varied pacing (fast story, slow key point)

**Time estimate:** 90 minutes for complete script

**Process:**

   - Questions to guest

**Podcast script writing tips:**
- More conversational than video script
- Longer stories (listeners have more attention)
- Sound effects marked (music, transitions)
- Guest interruptions noted
- Pacing slower (listeners process differently)

### WEDNESDAY 8:00am - 5:00pm: PRODUCTION DAY
**Goal:** Transform scripts into finished video and podcast assets

#### WEDNESDAY 8:00am-12:00pm: VIDEO PRODUCTION (4 hours)

**Video Recording Setup (30 min)**
```
CHECKLIST:
‚òê Camera/phone fully charged
‚òê Microphone positioned
‚òê Lighting adequate
‚òê Script printed or on screen
‚òê Recording location quiet
‚òê 15 min warm-up (speak lines aloud)
‚òê Test recording (check audio/video quality)
```

**Recording Process (90 min)**
```
1. Record INTRO segment (5 attempts, pick best)
   - Read script once first (don't film)
   - Record 3-4 takes
   - Note best take number

2. Record MAIN CONTENT (30 min across all insights)
   - One insight per recording session
   - Do 2-3 takes of each
   - Vary energy/tone slightly

3. Record OUTRO (5 attempts)
   - Same process as intro

4. Record B-ROLL (90 min)
   - Get supporting footage for each section
   - Let camera roll while you speak
   - Capture hands, workspace, visuals

TOTAL: 2 hours recording
```

**Video Editing (90 min - CapCut)**
```
1. Import all video clips into CapCut

2. Arrange timeline:
   - Intro segment
   - Insight 1 (best take)
   - Insight 2 (best take)
   - Insight 3 (best take)
   - Outro segment

3. Trim footage (remove errors, pauses)

4. Add graphics:
   - Intro title card (3 sec)
   - Main stat overlay (2 sec during first insight)
   - Text callouts (key points)
   - Outro slide with CTA (3 sec)

5. Add music:
   - Background music (royalty-free from CapCut library)
   - Fade in at intro (0.5 sec)
   - Lower volume during speech (-8dB)
   - Fade out at outro

6. Add captions:
   - Use CapCut auto-caption
   - Review and correct
   - Adjust timing
   - Color contrast (white on dark)

7. Export versions:
   - YouTube version: 1920√ó1080 (16:9) MP4
   - TikTok version: 1080√ó1920 (9:16) MP4
   - LinkedIn version: 1920√ó1080 (16:9) MP4
   - All at 1080p quality

TOTAL: 90 minutes editing
```

**Video Quality Check**
```
‚òê Audio levels consistent
‚òê Pacing matches script
‚òê Captions 100% accurate
‚òê Graphics support message
‚òê CTA clear and visible
‚òê No watermarks (or approved watermark)
‚òê Color correction applied
‚òê Aspect ratios correct
‚òê File sizes reasonable (<500MB for YouTube)
```

#### WEDNESDAY 12:00pm-1:00pm: LUNCH BREAK (60 min)

#### WEDNESDAY 1:00pm-4:00pm: PODCAST PRODUCTION (3 hours)

**Podcast Recording Setup (30 min)**
```
CHECKLIST:
‚òê USB microphone connected and tested
‚òê Audacity open and configured
‚òê Recording level set (-12dB on meter)
‚òê Audio interface recognized
‚òê Headphones working (listen while recording)
‚òê Script printed or accessible
‚òê Phone on silent
‚òê Door closed (minimize noise)
‚òê Test recording (30 sec, check quality)
```

**Recording Process (90 min)**
```
1. Read through entire script once (don't record)

2. Start recording

3. Read INTRO section (2 min)
   - Natural pace
   - Pause for breath
   - Stop and replay to check audio

4. Read BODY SEGMENT 1 (5 min)
   - Conversational tone
   - Change pace for emphasis
   - Pause longer for key points

5. Read BODY SEGMENT 2 (5 min)
   - Repeat process
   - Maintain energy

6. Read BODY SEGMENT 3 (5 min)
   - Repeat process
   - Build toward conclusion

7. Read OUTRO (2 min)
   - Warm, personal tone
   - Clear CTA
   - Thank you

TOTAL: ~20-25 minutes recording (plus breaks between segments)
```

**Podcast Editing (60 min - Audacity)**
```
1. Import audio file into Audacity

2. Clean audio:
   - Noise reduction (Audacity effect: Generate > Noise)
   - Normalize (-1.0dB target)
   - Equalize if needed (remove low rumble)

3. Edit timeline:
   - Remove long silences (>3 sec)
   - Remove mistakes/wrong takes
   - Trim start/end silence
   - Ensure flow

4. Add intro music:
   - Royalty-free podcast intro (20 sec)
   - Fade in 5 sec, fade out 3 sec
   - Keep intro before speech

5. Add outro music:
   - Royalty-free podcast outro (20 sec)
   - Fade in 3 sec before end
   - Fade out 5 sec after speech ends

6. Export:
   - MP3 format
   - 128 kbps quality
   - Mono or Stereo (mono is fine)
   - Filename: "[BlogTitle]-Episode.mp3"

TOTAL: 60 minutes editing
```

**Podcast Quality Check**
```
‚òê Audio clear (no background noise)
‚òê Levels consistent throughout
‚òê Pacing natural
‚òê Intro/outro music professional
‚òê File size reasonable (<100MB)
‚òê No dropped audio sections
‚òê Captions/transcript ready (optional)
‚òê Show notes prepared
‚òê Podcast cover art final
```

## TROUBLESHOOTING GUIDE

**If blog isn't done by Monday 2pm:**
- Don't delay week. Publish Tuesday morning instead
- Move all other content distribution back 1 day
- Repurpose happens same schedule, just shifted

**If you can't record video Wednesday:**
- Use blog images + text overlays in CapCut instead (takes 2 hours)
- Create video for next Thursday (publish following Friday)
- Don't skip video‚Äîjust delay 1 week

**If podcast audio quality is poor:**
- Re-record just the problematic sections (usually 5-10 min)
- Audacity will stitch together automatically
- Increase mic distance + reduce volume 2dB

**If email not reaching subscribers:**
- Check spam folder (might be filtered)
- Verify email addresses are correct in ConvertKit
- Send test to yourself first
- Check email preview in Buffer before scheduling

**If social posts not getting engagement:**
- Timing might be off (change posting time by 2 hours)
- Content might not resonate (note for next week)
- CTAs might be weak (test different CTA next post)
- Hashtags might be wrong (use trending hashtags)
