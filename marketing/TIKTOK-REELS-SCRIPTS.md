### HASHTAG POSTING RULES (Critical)

**TikTok Hashtag Best Practices:**
- Use 3-5 hashtags (algorithm penalizes hashtag spam)
- Place hashtags in caption (not comments)
- Mix high-volume (#ForTheKids) + niche (#PediatricCharity) tags
- Avoid hashtag repetition (use different mix each time)
- Test new hashtags weekly (track performance)

**Instagram Reels Hashtag Best Practices:**
- Use 10-15 hashtags (Instagram rewards comprehensive tagging)
- Mix: 3 high-volume (1M+ posts), 5 mid-volume (100K-1M), 5 niche (<100K)
- Place hashtags in caption OR first comment (both work)
- Use all 3 tiers for maximum discoverability
- Research similar accounts' hashtags (what works for them works for you)

**Prohibited Hashtags (Gospel V1.3 Compliance):**
```

   (We are NOT a charity, we are a for-profit platform)

   #TransparentCharity (Gospel V1.3)
```
