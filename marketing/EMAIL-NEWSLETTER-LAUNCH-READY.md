# SECTION 2: A/B TESTING FRAMEWORK

## Testing Methodology

### Principles
1. **Control vs. Variant:** Always test one "control" (proven approach) vs. one "variant" (new approach)
2. **Sample Size:** Minimum 1,000 recipients per variant for statistical significance
3. **Split:** 50/50 randomization (SendGrid A/B testing built-in)
4. **Duration:** Run test for 72 hours minimum
5. **Win Criteria:** Highest conversion rate wins (not just opens)

### Success Metrics by Email Type

**Welcome Sequence**
- Target open rate: 30%+
- Target click rate: 5%+
- Target conversion rate: 15%+ to Pro
- Win threshold: +2% improvement to declare winner

**Backer Thank You Sequence**
- Target open rate: 60%+
- Target click rate: 10%+
- Target referral link clicks: 20%+
- Win threshold: +3% improvement

**Re-engagement Sequence**
- Target open rate: 20%+
- Target click rate: 8%+
- Target reactivation rate: 10%+
- Win threshold: +2% improvement

**Monthly Impact Newsletter**
- Target open rate: 55%+
- Target click rate: 14%+
- Target dashboard visit: 25%+
- Win threshold: +3% improvement

---

## Testing Schedule

### MONTH 1-2: BASELINE (No Testing)
- Run all sequences as written (control versions)
- Collect baseline metrics for all emails
- Record: Open rate, click rate, conversion rate, bounce rate, spam complaints
- Purpose: Establish benchmark for comparison

### MONTH 3-4: SUBJECT LINE TESTS
- Test 2 subject lines per sequence (using A/B variants provided)
- Run tests one sequence at a time
- Document results in shared spreadsheet
- Keep winning subject lines, retire losers

**Test Order:**
- Week 1-2: Welcome Email 1 (most critical)
- Week 3-4: Welcome Email 5 (CTA email, high impact)
- Week 5-6: Backer Thank You Email 1 (conversion focus)
- Week 7-8: Re-engagement Email 1 (dormant user focus)

### MONTH 5-6: BODY CONTENT TESTS
- Keep winning subject lines from Month 3-4
- Test CTA placement (top vs. middle vs. bottom)
- Test message length (short version vs. detailed version)
- Test tone (technical vs. emotional vs. balanced)

**Test Structure:**
- Version A: Original body (control)
- Version B: Revised body (variant)
- Run on new cohort of users
- Winner replaces original

### MONTH 7+: CONTINUOUS OPTIMIZATION
- Implement winning combinations from months 3-6
- Start testing secondary email types (impact newsletter, referral sequence)
- Run seasonal variations (holiday messaging, Q4 giving season)
- Quarterly audits to ensure no decay in performance
