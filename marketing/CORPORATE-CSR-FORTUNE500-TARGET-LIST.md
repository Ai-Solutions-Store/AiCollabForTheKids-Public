## TARGET LIST FORMAT & PRIORITY SCORING

### LEAD QUALIFICATION MATRIX

Each prospect is ranked by:
- **CSR Budget** - Annual giving capacity (higher = better)
- **Matching Gift Adoption** - Current workplace giving maturity
- **Pediatric Alignment** - Health/kids focus in mission
- **Geographic Proximity** - Ease of volunteer day coordination
- **Tech Adoption** - Blockchain/transparency values alignment
- **Timeline** - Decision-making speed (Q1 budgets = faster)

**Score Formula:** (Budget × 0.35) + (Adoption × 0.25) + (Alignment × 0.20) + (Proximity × 0.10) + (Tech × 0.10) = Total Priority Score (0-100)

## OUTREACH SEQUENCING STRATEGY

### Phase 1: Weeks 1-2 (Research & Warm Intros)
**Target:** Tier 1 companies (Microsoft, Apple, Google, Salesforce, JPM)
- Research LinkedIn for CSR director
- Find mutual connections (alumni, board connections)
- Prepare warm intro requests
- Draft customized pitch for each

### Phase 2: Weeks 3-6 (Initial Outreach)
**Target:** First contact with 5 Tier 1 + 5 Tier 2 companies
- Send warm intro emails (if available)
- Cold email to CSR directors if no warm intro
- Include 1-page "FOR THE KIDS Overview"
- Goal: Schedule 15-minute discovery calls

### Phase 3: Weeks 7-10 (Discovery Calls)
**Target:** 5-10 qualified calls
- Listen for matching gift program gaps
- Identify volunteer day opportunities
- Understand budget cycles (Q4/Q1 budgets = faster decisions)
- Ask about ERG engagement possibilities

### Phase 4: Weeks 11-16 (Proposal Customization)
**Target:** 2-3 formal proposals
- Customize sponsor tier package
- Include company-specific case studies
- Propose pilot program (smaller scale)
- Set implementation timeline

### Phase 5: Weeks 17+ (Closing & Execution)
**Target:** 1-2 closed partnerships
- Negotiate terms
- Set up integration (Benevity/YourCause)
- Execute first volunteer day
- Generate press release + media coverage
