### Attribution Rules

**First-Touch Attribution (Primary):**
- First referral link clicked wins the commission
- Example: User clicks Affiliate A on Day 1, clicks Affiliate B on Day 10, subscribes on Day 15 â†’ Affiliate A gets credit

**Cookie Window:**
- Bronze: 30 days
- Silver: 60 days
- Gold/Platinum: 90 days
- Legend/Charity Partners: Lifetime (never expires)

**Last-Click Override (Same Session):**
- If user clicks different affiliate link within same browsing session â†’ Cookie updates to most recent
- Prevents accidental attribution to old clicks

**Manual Override:**
- If tracking fails due to system error â†’ Manual credit issued after review
- If customer deliberately bypassed link â†’ No credit

### Fraud Prevention

**Self-Referral Detection:**
- Email matching: Does subscriber email match referrer?
- IP matching: Does subscriber IP match referrer within 48 hours?
- Payment matching: Same credit card across multiple referrals?
- If detected â†’ Flag for review â†’ Confirmed fraud = account termination

**Click Farm Detection:**
- Abnormal click patterns (100+ clicks from same IP)
- Bot traffic detection (unusual user agents)
- Geographic clustering (all clicks from same city/country)
- Action: Flag affiliate, withhold payout pending investigation

**Conversion Rate Anomalies:**
- Normal rate: 3-10%
- If affiliate shows 60%+ conversion â†’ Suspicious (may indicate paid clicks)
- If affiliate shows <1% conversion â†’ Low quality traffic (re-engagement needed)

**Refund/Chargeback Tracking:**
- If customer refunds within 30 days â†’ Commission clawed back automatically
- If 20%+ of referrals request refunds â†’ Investigation for fraudulent traffic
- High chargeback rate â†’ Account termination
